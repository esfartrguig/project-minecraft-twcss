Configuration Minimale

CPU / Processeur Intel Core i3-3210 3.2 GHz/ AMD A8-7600 APU 3.1 GHz / Apple M1 ou équivalent GPU / Processeur graphique intégré Intel HD Graphics 4000 (Ivy Bridge) ou AMD Radeon R5 (Kaveri) avec OpenGL 4.4* GPU / Carte graphique dédiée Nvidia GeForce 400 Series ou AMD Radeon HD7000 avec OpenGL 4.4 Disque dur / HDD Au moins 1 Go pour le jeu de base, les cartes et d’autres fichiers RAM / Mémoire vive 2 Go OS Windows: Windows 7 et plus macOS: tout OS X 64 bits utilisant 10.9 Maverick ou plus récent Linux: toutes les distributions 64 bits modernes à partir de 2014 Réseau La connectivité Internet est nécessaire pour télécharger les fichiers Minecraft, après quoi la lecture hors connexion est possible.
